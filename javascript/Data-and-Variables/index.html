1. 데이터 타입 (Data type)

원시형(primitive type)

객체가 아니면서 메서드도 가지지 않는 데이터입니다.
모든 원시 값은 불변성 을 띄고 있습니다.

  • 불변성(immutability) : 원시 값은 변형할 수 없습니다. 변수를 할당하는 것과는 다른 개념입니다.

  • 종류 : 숫자, 문자열, 불리언, null, undefined, 심볼

객체형(reference type)

메모리에 값을 주소로 저장하고, 출력시 메모리 주소와 일치하는 값을 출력합니다.

  • 종류 : 배열, 함수, 날짜, 정규표현식, map, weakmap, set, weaksset 등

2. 데이터 할당

불변값

Code에 변수를 선언합니다.
변수 x를 선언하라는 명령을 받은 컴퓨터는 비어있는 메모리 하나를 확보하고 그 공간의 이름을 x라고 지정합니다.
선언한 변수 x에 "aaa" 문자열을 할당해보겠습니다.
비어있는 메모리 하나를 확보하고 "aaa"라는 문자열이 할당됩니다.
그 다음, 변수 x가 가르키고 있는 값의 주소를 "aaa"가 저장되어 있는 메모리 주소가 할당됩니다.
변수 x에 "bbb"라는 문자열을 재할당 해보겠습니다.
비어있는 메모리 하나를 확보하고 "bbb" 문자열을 할당됩니다.
변수 x가 가르키고 있는 주소가 @5002로 변경됩니다.
새로운 변수 y를 선언하고 "bbb"로 초기화해보겠습니다.
기존에 "bbb"가 할당되어 있는 메모리 주소를 가르키게 됩니다.
<Code>
let x;
x = "aaa";
x = "bbb";
y = "bbb";
<Memory>
@1001 : x =>
@1001 : x => @5001
@1001 : x => @5002
@1002 : y => @5002
@5001 : "aaa"
@5002 : "bbb"
